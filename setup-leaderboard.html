Leaderboard not showing data?

  • Check if SQL functions were created successfully
  • Verify you have solved at least one problem
  • Check browser console (F12) for errors
  • Ensure Supabase credentials in .env.local are correct

RPC error messages?

Don't worry! The system has automatic fallback - it will calculate the leaderboard from raw data if the RPC function fails.

-- Check if functions exist
SELECT routine_name 
FROM information_schema.routines 
WHERE routine_name IN ('get_leaderboard', 'get_user_rank');
